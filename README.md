# Aura Aesthetic Clinic

Landing page profesional para una clínica de estética integral. Diseñada para presentar servicios de belleza, tratamientos faciales y corporales, destacando la confianza y el profesionalismo del centro. Incluye secciones para detallar los servicios, testimonios de clientes, galería visual y llamadas a la acción claras para facilitar la reserva de turnos.

## 🚀 Tecnologías

Este proyecto está construido con:

- **[Astro](https://astro.build/)**: Framework web para la velocidad.
- **[Tailwind CSS](https://tailwindcss.com/)**: Estilos modernos y rápidos.
- **[Astro Sitemap](https://docs.astro.build/en/guides/integrations-guide/sitemap/)**: Para mejorar el SEO.

## 🧞 Comandos

Todos los comandos se ejecutan desde la raíz del proyecto:

| Comando | Acción |
| :--- | :--- |
| `npm install` | Instala las dependencias |
| `npm run dev` | Inicia el servidor de desarrollo en `localhost:4321` |
| `npm run build` | Construye el sitio para producción en `./dist/` |
| `npm run preview` | Previsualiza la construcción localmente |
| `npm run astro ...` | Ejecuta comandos de la CLI de Astro |

## 📁 Estructura del Proyecto
